关于 XMUSIC

XMUSIC 是一个去中心化的音乐投资平台，让每个独立歌手都能像创业公司一样运作自己的音乐事业。

我们的使命

让音乐投资变得简单、透明、 accessible。我们相信每个有才华的音乐人都应该有机会获得资金支持，让每个热爱音乐的人都能成为音乐事业的股东。

核心功能

  • 项目融资 - 歌手创建融资项目，设定目标和份额分配
  • 投资份额 - 投资人使用 USDC 购买项目份额，成为股东
  • 收益分配 - 智能合约自动按比例分配收益给所有股东
  • 治理投票 - 份额代币代表治理权，参与项目重大决策

技术架构

XMUSIC 采用 Web3 技术栈：

  • 前端：Next.js + React + Tailwind CSS
  • 智能合约：Solidity + OpenZeppelin
  • 存储：IPFS 去中心化存储
  • 支付：USDC 稳定币
