## 一、AI算力/光模块方向：腾景科技（688195.SH）

### 1. 公司基本面

**腾景科技**是国内精密光学元组件龙头企业，成立于2013年，2021年于上交所科创板上市。公司专业从事各类精密光学元组件、光纤器件、光测试仪器研发、生产和销售，是光通信产业链上游的核心供应商。

**核心数据（截至2026年2月）：**
- 当前股价：约175元（近期波动区间164-209元）
- 总市值：约219亿元
- 总股本：1.29亿股
- 市盈率（TTM）：约282倍
- 市净率：约22.71倍

### 2. 技术壁垒

**六大核心技术平台：**
1. **光学薄膜类技术** - 纳米级光学镀膜工艺
2. **精密光学类技术** - 高精度光学元件加工
3. **模压玻璃非球面类技术** - 非球面透镜成型
4. **光纤器件类技术** - 光纤耦合与器件封装
5. **衍射光学类技术** - 衍射光栅设计与制造
6. **精密光学机械与电子集成类技术** - 光机电一体化

**核心竞争优势：**
- 产品精度达**纳米级**，技术门槛极高
- 通过EN9100:2018航空航天质量管理体系认证
- 国家级专精特新"小巨人"企业
- 拥有福建省院士专家工作站和企业技术中心
- 研发团队含5名博士、20名硕士

### 3. 客户情况

**光通信领域客户：**
- **谷歌**：通过Coherent间接供应TPU集群的OCS（光链路交换）系统
- **Lumentum**、**Finisar**（全球顶级光模块厂商）
- **光迅科技**、**苏州旭创**（国内光模块龙头）
- **华为**、**中兴**（国内通信设备巨头）

**光纤激光领域客户：**
- **nLIGHT**（世界领先高功率半导体激光器厂商）
- **R客户**（全球知名光纤激光器企业）

**最新进展：**
- 2026年1月收到某客户C的子公司关于二维准直器阵列的订单
- 产品通过谷歌、微软、英伟达等科技巨头的供应链认证

### 4. 财务数据

**2025年前三季度业绩：**
| 指标 | 数值 | 同比变动 |
|------|------|----------|
| 营业收入 | 4.25亿元 | +28.11% |
| 归母净利润 | 6380万元 | +15.0% |
| 毛利率 | 39.12% | +0.79pct |
| 经营现金流 | 9459万元 | 健康 |

**单季度表现（2025Q3）：**
- 营业收入：1.62亿元，同比增长34.83%
- 净利润同比增长20.11%

### 5. 催化剂

1. **谷歌OCS光交换机放量**：谷歌TPU集群持续扩张，OCS作为核心组件需求爆发
2. **英伟达LPU新架构**：LPU集群与OCS协同光互联带来增量需求
3. **1.6T光模块升级**：2026年1.6T光模块需求缺口预计达数百万只
4. **硅光技术渗透**：预计2026年硅光技术占比超50%，公司作为上游核心供应商受益
5. **国产替代加速**：华为等国内巨头供应链本土化趋势

### 6. 风险因素

- **估值偏高**：当前PE超280倍，需业绩持续高增消化估值
- **客户集中度风险**：对谷歌等大客户依赖度较高
- **技术迭代风险**：光通信技术快速迭代，需持续研发投入
- **行业竞争加剧**：光模块厂商向上游延伸可能带来竞争

## 二、固态电池/钠电池方向：当升科技（300073.SZ）

### 1. 公司基本面

**当升科技**是全球锂电正极材料行业龙头企业，央企背景（控股股东为矿冶集团），业务涵盖多元材料、磷酸（锰）铁锂、钠电材料、钴酸锂和多元前驱体等正极材料的研发、生产和销售。

**核心数据（截至2026年2月）：**
- 当前股价：约56-57元
- 总市值：约288-309亿元
- 市盈率（TTM）：约45-50倍
- 市净率：约2.5-3倍

### 2. 技术壁垒

**固态电池材料布局：**
- **双相复合高能量固态正极材料**技术路线
- **氧化物电解质**规模化产线已就绪
- **硫化物电解质**规模化产线已就绪
- 固态锂电关键材料已批量导入头部客户

**钠离子电池布局：**
- 全面覆盖**层状氧化物**、**聚阴离子**等钠电技术路线
- 产品寿命长、能量密度高
- 已获下游客户高度认可并逐步量产

**研发实力：**
- 新能源材料研究院、锂电新材料研究院、锂电材料工程技术研究院三大研发机构
- 最近三年累计研发投入16.38亿元，研发占比3.7%
- 固态电池正极研发成果入选**中央企业科技创新成果推荐目录**

### 3. 客户情况

**固态电池客户（已批量导入）：**
- **辉能科技**（台湾固态电池龙头，2026年2月签署战略合作协议）
- **清陶能源**（国内固态电池领军企业）
- **卫蓝新能源**（蔚来汽车固态电池供应商）
- **赣锋锂电**（全球锂业龙头旗下电池企业）
- **中汽新能**等

**海外动力电池客户：**
- **LGES**、**SK on**、**三星SDI**、**Murata**
- 进入**大众**、**现代**、**戴姆勒**、**宝马**、**奔驰**等国际主流车企供应链

**国内动力电池客户：**
- **比亚迪**、**中创新航**、**亿纬锂能**、**蜂巢能源**
- 为**理想**、**小鹏**、**长城**、**广汽**、**上汽**等国内高端车企配套

### 4. 财务数据

**2025年前三季度业绩：**
| 指标 | 数值 | 同比变动 |
|------|------|----------|
| 营业收入 | 73.99亿元 | +33.92% |
| 归母净利润 | 5.03亿元 | +8.30% |
| 扣非净利润 | 4.8亿元 | +8%左右 |

**单季度表现（2025Q3）：**
- 营业收入：29.67亿元，同比增长49.54%
- 归母净利润：1.92亿元，同比增长8.02%
- 扣非净利润：1.36亿元，同比增长29.36%

**2025年全年业绩预告：**
- 预计归母净利润：6.0-6.5亿元
- 同比增长：27.16%-37.76%

### 5. 催化剂

1. **固态电池量产元年**：2026年被视为固态电池量产元年，公司作为正极材料核心供应商将率先受益
2. **头部客户绑定**：与辉能、清陶、卫蓝等固态电池头部企业深度合作
3. **钠电池放量**：储能+两轮电动车市场推动钠离子电池需求爆发
4. **海外客户拓展**：深度融入国际高端车企及一线品牌动力电池产业链
5. **央企背景优势**：矿冶集团控股，在资源整合、技术合作方面具备优势

### 6. 风险因素

- **行业竞争加剧**：正极材料产能过剩，价格战风险
- **技术路线不确定性**：固态电池技术路线尚未完全定型
- **原材料价格波动**：钴、镍、锂等金属价格波动影响成本
- **海外政策风险**：欧美新能源政策变化可能影响出口

## 三、人形机器人方向：五洲新春（603667.SH）

### 1. 公司基本面

**五洲新春**是一家集研发、制造、营销服务一体化的综合型企业集团，深耕精密制造技术二十多年，是国内少数涵盖精密锻造、制管、冷成形、机加工、热处理、磨加工、装配的轴承、精密零部件全产业链企业。

**核心数据（截至2026年2月）：**
- 当前股价：约86-91元（波动较大）
- 总市值：约310-330亿元
- 动态市盈率：约250-360倍（估值偏高）
- 市净率：约11倍

### 2. 技术壁垒

**全产业链优势：**
- 自主掌控**材料研发**、**精密成型**（冷辗/热处理工艺）、**智能化磨装**
- 实现全链条质量可控、成本优化和技术迭代闭环

**核心产品技术：**
- **轴承产品**：精密轴承制造技术国内领先
- **丝杠产品**：行星滚柱丝杠、微型滚珠丝杠技术突破
- **精密机械零部件**：高精度机加工能力

**研发平台：**
- 国家企业技术中心
- CNAS国家认证实验室
- 院士专家工作站
- 浙江省高端轴承重点企业研究院
- 航空滚动轴承浙江省工程研发中心

### 3. 客户情况

**特斯拉Optimus供应链：**
- 根据2025年12月特斯拉Optimus Gen3量产审厂结果，五洲新春被确定为**7家核心供应商之一**
- 主要负责**轴承**、**行星滚柱丝杠**等核心零部件供应
- 为Gen3丝杠提供定制化服务

**其他机器人客户：**
- **智元机器人**：华为供应链体系
- **小米机器人**：国内人形机器人新势力
- **优必选**：国内人形机器人龙头企业

**传统优势客户：**
- 汽车零部件：国内外一流汽车厂商
- 风电轴承：国内主流风电设备商

### 4. 财务数据

**2025年前三季度业绩：**
| 指标 | 数值 | 同比变动 |
|------|------|----------|
| 营业收入 | 26.61亿元 | +7.60% |
| 归母净利润 | 9848万元 | +0.25% |
| 扣非净利润 | 9136万元 | +7.00% |
| 经营现金流 | 1.41亿元 | 健康 |

**单季度表现（2025Q3）：**
- 营业收入：7.67亿元，同比增长6.33%
- 归母净利润：2319万元，同比增长0.07%

**定增募投项目（2025年）：**
- 募资10亿元用于人形机器人零部件产能建设
- 产能设计：98万套行星滚柱丝杠、210万套微型滚珠丝杠、7万组通用机器人专用零部件

### 5. 催化剂

1. **特斯拉Optimus量产**：2025年计划量产1万台，2026年扩产至5万台，供应链订单即将落地
2. **人形机器人元年**：2026年被视为人形机器人量产和商业化元年
3. **定增产能释放**：10亿募资投向机器人零部件，产能爬坡后业绩弹性大
4. **国产机器人崛起**：智元、小米、华为等国内厂商加速布局
5. **丝杠国产替代**：高端丝杠进口替代空间大，公司产品技术突破

### 6. 风险因素

- **估值严重偏高**：当前PE超250倍，股价已透支未来数年业绩
- **业绩兑现不确定性**：机器人订单尚未形成实质收入
- **竞争加剧**：三花智控、拓普集团等强敌环伺
- **技术迭代风险**：人形机器人技术路线仍在快速迭代
- **股价波动大**：概念炒作成分较高，波动风险大
