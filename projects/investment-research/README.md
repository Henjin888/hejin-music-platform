# 投资研究领域档案

## 投资理念
**核心策略**：寻找未来5-10年爆发领域中的1-2元低价股
**目标回报**：50-100倍
**时间框架**：长期持有（3-10年）

---

## 重点追踪领域

### 🔥 第一梯队（高确定性）

#### 1. 人工智能产业链
**爆发时间**：正在进行中，未来5年持续
**细分领域**：
- AI芯片（边缘计算、推理芯片）
- AI应用（企业级SaaS、垂直行业AI）
- AI基础设施（数据中心、光模块）
- AI Agent平台

**选股逻辑**：
- 市值<100亿
- 有核心技术或数据壁垒
- 营收增速>30%
- 估值合理（PE<50或PS<10）

#### 2. 新能源（储能+氢能）
**爆发时间**：2025-2030
**细分领域**：
- 固态电池
- 钠离子电池
- 氢能产业链（制氢、储氢、运氢、用氢）
- 新型储能（压缩空气、液流电池）

**选股逻辑**：
- 技术路线清晰，有量产能力
- 绑定头部客户
- 成本控制能力强

#### 3. 生物科技（基因治疗+合成生物学）
**爆发时间**：2025-2035
**细分领域**：
- 基因编辑（CRISPR应用）
- 细胞治疗（CAR-T等）
- 合成生物学（替代化工、生物制造）
- mRNA技术平台

**选股逻辑**：
- 管线丰富，有进入临床的产品
- 技术平台可扩展
- 与Big Pharma有合作

---

### 🔥 第二梯队（中等确定性）

#### 4. 太空经济
**爆发时间**：2028-2035
**细分领域**：
- 卫星互联网（低轨星座）
- 火箭发射服务
- 太空资源开发
- 太空旅游

**选股逻辑**：
- 有实际订单或发射记录
- 技术验证通过
- 政府/军方背书

#### 5. 量子计算
**爆发时间**：2030-2040
**细分领域**：
- 量子计算机硬件
- 量子软件/算法
- 量子通信
- 量子传感

**选股逻辑**：
- 技术路线被验证
- 有商业化场景
- 与科研院所合作紧密

#### 6. 人形机器人
**爆发时间**：2027-2035
**细分领域**：
- 核心零部件（减速器、电机、传感器）
- 整机制造
- 机器人大脑（AI算法）

**选股逻辑**：
- 进入特斯拉/小米/华为供应链
- 技术参数领先
- 量产能力验证

## 每日选股检查清单

### 基础筛选
- [ ] 股价1-2元（或市值<50亿）
- [ ] 日均成交额>1000万（确保流动性）
- [ ] 无ST风险
- [ ] 主营业务清晰

### 深度分析
- [ ] 所属行业是否在高增长赛道
- [ ] 是否有核心技术/资源/牌照壁垒
- [ ] 管理层背景和能力
- [ ] 财务状况（现金流、负债率）
- [ ] 催化剂（订单、政策、技术突破）

### 风险评估
- [ ] 最大回撤承受能力
- [ ] 仓位分配建议
- [ ] 止损位设置
