## 执行规则

1. **严格按时汇报**：每个整点输出进度报告
2. **优先级排序**：投资推送 > 创业项目 > AI学习 > 趋势研究
3. **质量优先**：宁可少做，也要保证质量
4. **及时沟通**：遇到阻塞立即汇报
5. **文档沉淀**：所有产出写入对应档案

### 创业项目
- 代码提交频率

- Bug修复速度

### AI学习
- 新工具测试数量
- 学习笔记质量
- 项目应用转化率
