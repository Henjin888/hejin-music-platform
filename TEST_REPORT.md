## 🛠️ 修复详情

### 1. index.html - DOM XSS 和代码结构修复

**问题:**
- 存在重复的 `expandSidebar` 和 `collapseSidebar` 函数定义
- 事件监听器在DOMContentLoaded外部重复定义，导致潜在的多重绑定
- 内联事件处理程序使用 `onclick` 存在XSS风险
- 缺少错误处理机制

**修复:**
- 合并DOMContentLoaded事件处理程序
- 将侧边栏事件监听器移到DOMContentLoaded内部
- 添加try-catch错误处理
- 统一使用 `window.expandSidebar` 暴露函数
- 添加元素存在性检查

### 2. discover.html - 变量未声明修复

**问题:**
- 使用未声明的 `currentLang` 变量
- 缺少语言显示元素的更新
- 初始化逻辑缺少错误处理

**修复:**
- 添加 `let currentLang = lang;` 声明
- 添加语言显示元素ID和更新逻辑
- 使用try-catch包装初始化代码

### 3. xmusic.js - 错误处理增强

**问题:**
- 货币转换和翻译函数缺少边界情况处理
- 缺少函数文档注释
- 可能导致运行时错误

**修复:**
- 添加空值检查和默认值
- 添加try-catch块
- 添加JSDoc风格的中文注释
- 添加类型检查

### 4. login.html - 输入验证增强

**问题:**
- 硬编码验证码"123456"存在安全风险
- 缺少手机号格式验证
- 缺少验证码格式验证

**修复:**
- 移除硬编码验证码，添加TODO注释
- 添加手机号正则验证: `/^1[3-9]\d{9}$/`
- 添加验证码正则验证: `/^\d{6}$/`
- 添加详细的错误提示
